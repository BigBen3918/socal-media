## Demo

https://github.com/galaxy-foundation/desocial_media.git

## How to use?

1. Download or clone this repo.

2. Install dependencies.

```js
npm install
// or
yarn install
```

3. Run project on iOS / Android.

```js
 npm run android // npm run android
 // or
 yarn android // yarn android
```
